# Home Assistant Documentation

Welcome to your Home Assistant documentation site.

This page is rendered from `/config/docs/index.md`. Edit it to describe
your setup — automations, dashboards, hardware notes, anything you want
to keep close to your configuration.

## How this add-on works

- `/config/docs/index.md` is the landing page.
- Every other `.md` file under `/config/docs/` becomes a page in the
  navigation; sub-directories become sections.
- The site rebuilds automatically on save; just refresh your browser.

See **Getting started** in the sidebar for syntax and feature examples.
